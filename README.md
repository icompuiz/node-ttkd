TTKD Student Management Application 
==============


###Major Technologies Used
- NodeJS
- MongoDB
- Mongoose ODM
- AngularJS
- - Angular UI Router
- Twitter Bootstrap
- LoDash
- Node Async
- Grunt
- Bower
- Karma

###Installation

####Prerequisites
	1. MongoDB
	2. NodeJS ~0.10.28

####Steps
	1. Clone/Fork this repository
	2. Execute
		```
		$ npm install
		```
	3. To run with grunt
		```
		$ grunt serve 
		```
	4. To run without grunt
		```
		node server.js
		```
